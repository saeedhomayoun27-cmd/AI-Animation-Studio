# 🎬 AI Animation Studio

An open-source AI animation production studio based on ComfyUI.

## Features

- Text to Image
- Image to Video
- Text to Video
- Storyboard Management
- Character Consistency
- ComfyUI Integration
- Google Colab Support

## Project Structure

- workflows : ComfyUI workflows
- models : AI models
- loras : Character and style models
- controlnets : Control models
- prompts : AI prompts
- storyboard : Story planning
- outputs : Generated files
